# 图书管理系统 python775

本项目是一款基于Python + Django + SQLite的图书管理系统，旨在提供便捷的图书管理、借阅管理等功能，通过数据可视化图表，帮助用户更直观地了解图书流通情况。

## 技术栈

- 后端：Python 3
- 前端：Django框架
- 数据库：SQLite
- 数据可视化：Django Chart.js

## 功能模块

### 登陆功能

- 用户身份验证
- 权限管理

### 图书管理

- 图书信息录入
- 图书查询
- 图书借阅状态更新

### 借阅管理

- 借阅申请
- 归还处理
- 借阅历史查询

### 个人管理

- 修改个人信息
- 查看借阅记录

### 会员管理

- 会员信息管理
- 会员借阅权限设置

### 数据可视化图表

- 图书借阅统计
- 会员借阅频次分析

## 系统角色

- 系统管理员：负责系统维护、用户权限管理、图书信息管理
- 普通用户：可进行图书查询、借阅、归还等操作

## 运行环境

- Python 3.x
- Django 2.x
- SQLite

## 部署步骤

1. 安装Python 3.x
2. 安装Django 2.x
3. 安装SQLite数据库
4. 克隆项目代码
5. 导入数据库结构
